# Assets and App Name (Selorg Packman)

## App name (no hyphen)
- **Display name:** Selorg Packman (Expo Go, Web title, Android app label, iOS app label, Play Store, App Store).
- **Slug:** `selorgpackman` (used by Expo/EAS; no hyphen).

## Icon and splash assets
All paths are relative to `frontend/`. Stored in `assets/images/`:

| File | Purpose | Spec |
|------|--------|------|
| **splash-icon.png** | Logo image; native splash screen (centered, solid background) | 1024×1024 PNG |
| **icon.png** | Main app icon (Expo, Android, iOS, stores) | 1024×1024 PNG |
| **adaptive-foreground.png** | Android adaptive icon foreground | 512×512 PNG |
| **adaptive-background.png** | Android adaptive icon background (solid) | 1024×1024 PNG |
| **favicon.png** | Web favicon | 64×64 PNG |

- Splash uses `resizeMode: "contain"` (logo centered, not stretched) and solid `backgroundColor: "#2f8f3e"`.
- Store-specific sizes are generated by EAS Build from these files.

## Regenerating from logo
After replacing `splash-icon.png` with the official logo, run:

```bash
cd frontend
npm run generate-icons
```

This generates `icon.png`, `adaptive-foreground.png`, `adaptive-background.png`, and `favicon.png` from `splash-icon.png`.
